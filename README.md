# BackEnd
## Project setup
```
npm install
```
## Create .env 
```
Create config.js file under BackEnd folder
MONGO_DB_URL=<mongoDB URL>
SECRET_KEY=<secret key for JWT>
```
### Compiles and run
```
node index.js
```
